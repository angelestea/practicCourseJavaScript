Juniors

Se tomó como base vacantes para desarrolladores juniors en el área full stack. En dólares estadounidenses.

Se usó como análisis la base de datos adjunta en los archivos .js del proyecto. Para este análisis se usó 8 vacantes. Cabe resalatar también que el análisis fue realizado en base a puestos en países como Ecuador, España, Argentina y anglo parlantes.

En conclusión existe un basta gama de salarios para un desarrollador principiante. Sin embargo, como dato adicional, es de resaltar que los salarios más altos de este estatus son precisamente para puestos requeridos en el idioma inglés como medio comunicativo.
